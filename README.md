# 🦞 OpenClaw Memory Share

龙虾记忆分享平台 - 分享你的AI使用心得、技巧和灵感

## 功能特性

- ✅ 用户注册/登录 (JWT认证)
- ✅ 发布记忆 (支持Markdown)
- ✅ 一键复制内容
- ✅ 点赞、收藏、评论
- ✅ 扁平化UI设计
- ✅ 响应式布局

## 技术栈

**前端:**
- React 18
- Vite 5
- Tailwind CSS
- React Router
- Lucide Icons

**后端:**
- Node.js
- Express
- sql.js (SQLite)
- JWT认证
- bcryptjs

## 快速开始

### 后端
```bash
cd backend
npm install
node server.js
```

### 前端
```bash
cd frontend
npm install
npm run dev
```

## 访问地址

- 前端: http://localhost:8888
- 后端API: http://localhost:8282/api

## 部署

提供systemd服务配置，支持开机自启和崩溃自动重启。
