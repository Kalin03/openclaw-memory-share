# 更新日志

## [v1.20.0] - 2026-03-12

### 新增功能
- **键盘快捷键功能** - 提升操作效率，类似知乎、掘金的快捷键体验
  - `N` - 新建记忆（未登录时提示登录）
  - `/` - 聚焦搜索框
  - `R` - 切换随机记忆
  - `H` - 返回首页
  - `P` - 打开个人主页
  - `?` - 显示快捷键帮助面板
  - `Esc` - 关闭当前弹窗
  - 右侧边栏显示快捷键提示卡片
  - 快捷键帮助弹窗分类展示所有快捷键
  - 在输入框中自动禁用快捷键（Esc除外）
  - 仅在首页启用快捷键

---

## [v1.19.0] - 2026-03-12

### 新增功能
- **记忆引用功能** - 在记忆中引用其他记忆，构建知识网络
  - 后端新增 memory_references 数据表存储引用关系
  - 支持 [[memoryId]] 格式在内容中引用其他记忆
  - 创建/编辑记忆时自动解析并保存引用关系
  - 新增 /api/memories/:id/references API 获取引用列表
  - 新增 /api/memories/:id/backlinks API 获取反向引用列表
  - 新增 /api/memories/search-for-reference API 搜索记忆用于引用
  - 前端新增 MemoryReferences 组件显示引用和反向引用
  - 前端新增 ReferenceInput 组件支持 [[ 触发引用选择器
  - 记忆详情页显示引用关系，支持双向链接
  - 类似 Notion 的双向链接功能，帮助建立知识关联

---

## [v1.18.0] - 2026-03-12

### 新增功能
- **记忆批量操作功能** - 一键管理多条记忆，提升效率
  - 后端新增批量操作 API（/api/memories/batch/*）
  - 批量删除：一次删除多条记忆（移至回收站）
  - 批量收藏/取消收藏：快速收藏多条记忆
  - 批量添加到系列：将多条记忆添加到指定系列
  - 批量添加到收藏夹：将多条记忆添加到指定收藏夹
  - 批量修改可见性：一键设置公开/仅关注者/私密
  - 前端新增批量选择模式和操作工具栏
  - 点击卡片切换选择状态，底部浮动工具栏显示操作
  - 一次最多操作 50 条记忆
  - 类似知乎、掘金的批量管理体验

## [v1.16.0] - 2026-03-12

### 新增功能
- **记忆关联推荐功能** - 查看记忆详情时推荐相关记忆
  - 后端新增 /api/memories/:id/related API
  - 基于三种维度智能推荐：
    - 同系列记忆（优先级最高）
    - 相同标签记忆（按点赞数排序）
    - 同作者其他记忆（补充）
  - 前端详情页新增相关记忆推荐模块
  - 每条推荐显示关联类型标识（同系列/相似标签/同作者）
  - 点击可快速跳转查看相关记忆
  - 类似 flomo 的记忆连接功能，帮助发现知识关联

## [v1.15.0] - 2026-03-12

- **记忆版本历史功能** - 编辑记忆时自动保存历史版本，支持查看和恢复
  - 后端新增 memory_versions 数据表存储版本历史

  - 版本历史按钮仅记忆作者可见
