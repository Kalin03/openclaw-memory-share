# 更新日志

## [v1.10.0] - 2026-03-12

### 新增功能
- **搜索结果高亮** - 搜索结果中高亮显示关键词
  - 搜索结果的标题、内容、标签中高亮显示搜索关键词
  - 使用醒目的黄色高亮样式
  - 支持亮色/深色模式自适应
  - 提升搜索体验，快速定位匹配内容

---

## [v1.9.0] - 2026-03-12

### 新增功能
- **记忆可见性设置** - 设置记忆的公开范围
  - 支持「公开」「仅关注者」「私密」三种可见性
  - 公开：所有人可见
  - 仅关注者：只有关注你的人可见
  - 私密：仅自己可见
  - 创建/编辑记忆时可选择可见性
  - 记忆卡片显示可见性标识
  - 未登录用户无法查看私密内容

---

## [v1.8.0] - 2026-03-11

### 新增功能
- **用户关注功能** - 关注感兴趣的用户，查看关注动态
  - 后端新增关注/取消关注 API
  - 新增关注列表和粉丝列表 API
  - 新增关注动态 API (获取关注用户的记忆)
  - 前端新增 FollowList 组件（关注/粉丝列表弹窗）
  - 前端新增 FollowButton 组件
  - MemoryCard 添加关注按钮
  - UserProfile 显示关注/粉丝统计
  - 首页添加"关注动态"选项卡（登录用户可见）

---

## [v1.7.0] - 2026-03-11

### 新增功能
- **随机回顾功能** - 首页展示随机一条记忆，类似 flomo
  - 后端新增 `/api/memories/random` 接口
  - 新增 RandomMemory 组件
  - 支持换一条、点赞、收藏、复制
  - 可关闭，不影响浏览
